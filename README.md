# [Windows] 右クリックの「送る」メニューに機能追加

## 機能
* ファイルのパスをコピー
	* Markdown書式
	* Hiki(Wikiクローン)書式
* Hash計算
	* MD5
	* SHA256
	* SHA512

## セットアップ
1. このリポジトリをCloneする
1. `shell:sendto`（「送る」フォルダ）を開く（以下からお好きな方法で）
	* ファイル名を指定して実行（<kbd>Windows</kbd> + <kbd>R</kbd>）
	* エクスプローラー（<kbd>Windows</kbd> + <kbd>E</kbd>）
1. 「送る」フォルダに使いたいcmdファイルのショートカットを作成  
	※  ショートカット名は自由に変更可
